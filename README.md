
# FEN Tool

Convert Forsyth-Edwards Notation (FEN) strings to PNG images.

## Run Locally

Clone the project

```bash
  git clone https://github.com/Zeigst/fen_tool
```

Go to the project directory

```bash
  cd fen_tool
```

Run with Docker

```bash
  docker compose up --build
```
